iosDateModule Module

Description

Get the dates according the region format of your iOS device.

Accessing the iosDateModule Module

To access this module from JavaScript, you would do the following:

var iosdatemodule = require("com.dateFormatter");

The iosdatemodule variable is a reference to the Module object.

Usage

var iosdatemodule = require('com.dateFormatter');

//Pass fully qualified date string in the method//

var systemFormattedDate = iosdatemodule.getSystemFormattedDate('2014-12-29T16:17:26+05:30');

OR

// You can use javascript open source Moment.js Library to get the date in the above format//

var moment = require('alloy/moment');

var systemFormattedDate = iosdatemodule.getSystemFormattedDate(momemt('YOUR DATE STRING').format());

Ti.API.info('System formatted date as per the device region :' + systemFormattedDate);
